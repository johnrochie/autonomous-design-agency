## Tech Stack
- **Frontend:** Next.js 16 + TypeScript
- **Styling:** Tailwind CSS v4
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Payments:** Stripe Connect (integration ready)
- **Deployment:** Vercel

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```

### Deploy
Connected to Vercel - deployments triggered on push to main branch.

## Database Schema
See `supabase/schema.sql` for complete database schema including:
- 9 tables with RLS policies
- Real-time subscriptions for messages
- Auto-updating timestamps
